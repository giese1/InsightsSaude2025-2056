# InsightsSaude2025-2056

Este é um projeto na area de ciencia de dados

O dataset contém dados de exame de sangue em 2025, e varios insights desses exames.

Os atributos desse dataset são:

- Pontuação média de saude por pais
- risco médio de doenças por pais
- Marcadores biologicos mais frequentes por pais

Foi usado varios modelos de machine learning e deep learning neste projeto

O projeto tambem contém analise exploratoria dos dados

Fonte: https://www.kaggle.com/datasets/kantesti/global-blood-test-health-insights-2025-2026/data/code
